上个月我家门口又收到了一个神秘的盒子。

不是快递，也不是朋友送的礼物。打开一看，里面塞满了各种稀奇古怪的包装——有的上面写着我看不懂的文字，有的包装风格一看就不是超市货架上能找到的那种。

我女儿凑过来，眼睛都亮了："妈妈，这些是从哪来的？"

"巴西。"我说，"这个月的主题是巴西。"

这就是我订阅TryTreats零食盒的日常。每个月，这个盒子会从世界某个角落带来一堆你根本想不到的零食。上个月是马来西亚的斑兰蛋糕（老实说，有点干），再上个月是意大利的巧克力威化（这个真不错），再往前是哥伦比亚的咸味薯片。

说实话，刚开始我也半信半疑。毕竟市面上这种"全球零食"的噱头太多了，打开一看不是国产贴牌就是超市打折货。但TryTreats这家确实不一样——它每个月真的会换一个国家，而且里面的零食大部分在美国超市根本买不到。

## 每个月都是新鲜的冒险

TryTreats的特色就是"每月一个国家"。你永远不知道下个月会收到哪个国家的零食，这种不确定性反而成了最大的乐趣。

盒子里不只是零食，还会附带一张明信片、一些当地的小知识和食谱。有次收到日本主题的盒子,里面有张明信片介绍日本的茶道文化，还教你怎么做抹茶甜点。虽然我没真的去做，但看着也挺有意思的。

他们覆盖的范围很广：亚洲、欧洲、非洲、南美、北美、澳洲——基本上全球六大洲都有涉及。如果你不是美国人，还可以给他们发邮件，要求不要发你所在地区的零食。比如日本用户可以要求避开日本主题，这样就能更多地体验其他国家的新鲜玩意儿。

## 说说我踩过的坑

当然，不是每个月都完美。

马来西亚那期我就不太喜欢。斑兰蛋糕干得要命,巧克力面包也没什么特色，整体偏淡。但这也正常——零食这东西本来就看个人口味，我不喜欢的可能别人觉得还行。

还有一次收到一盒子椒盐脆饼干。我当时就想，全世界哪里都有椒盐饼干啊，这算什么国际零食？后来才知道那是德国传统款,味道确实跟超市的不一样。所以有时候你以为踩雷了，其实只是对那个国家的饮食文化不够了解。

包装偶尔会有点乱。有一次盒子到的时候有个小袋子被压扁了，不过零食本身没坏。这种情况其实挺少见的，而且他们客服响应很快——发个邮件基本24小时内就会回复。

## 值不值得订？

如果你是那种"超市货吃腻了"或者"想尝试新东西"的人，TryTreats挺适合的。

它不是那种高端精致路线，不会给你一大堆包装华丽但味道平庸的玩意儿。它走的是"真实"路线——就是把世界各地普通人日常吃的零食送到你手上。有些可能包装简陋，但味道确实有特色。

我最喜欢的还是那种"发现感"。你永远不知道下个月会吃到什么，也不知道哪个会成为你的新宠。就像开盲盒，但比开盲盒实用——至少能吃。

而且这个盒子真的很适合家庭聚会或者朋友来玩的时候拿出来分享。我有次开Party，把上个月收到的零食摆了一桌，大家边吃边猜这是哪个国家的，气氛特别好。

## 怎么订？

👉 [直接访问TryTreats官网](https://www.trytreats.com)，选择你想要的套餐类型和付款周期就行。

他们支持随时取消订阅，不会强制绑定。如果某个月不想要了，在每月25号之前取消就不会扣下个月的费用。年付计划的截止日期是每月20号。

对了，如果你不是美国用户，可能需要支付额外的国际运费。美国境内是免运费的，这点还挺良心。

配送一般用USPS，美国境内大概几天到两周不等。国际配送时间会长一些，1到4周都有可能，取决于你所在的国家和海关速度。

## 写在最后

TryTreats不会让你变成零食专家，也不会突然让你的生活变得多精彩。

它就是每个月给你一个小惊喜，让你在平淡的日子里多点期待。有时候打开盒子会失望，有时候会惊喜，但大部分时候你会觉得"嗯，还不错"。

如果你也厌倦了超市货架上那些老面孔，想试试别的，可以给自己订一个试试。反正第一个月不喜欢随时可以退。

就这样，我下个月的盒子应该快到了。这次不知道是哪个国家，期待一下。
